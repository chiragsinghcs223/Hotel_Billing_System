# 🏨 Hotel Billing System

A simple console-based **Hotel Billing System** developed in **Java** that calculates hotel bills based on room stay duration and additional services used by customers.

## 📌 Features

- Calculate room charges based on the number of days stayed.
- Add optional service charges:
  - Food Service
  - Laundry Service
  - Both Services
  - No Additional Services
- Generate the final bill with detailed breakdown.
- User-friendly menu-driven interface.

## 🛠️ Technologies Used

- Java
- Scanner Class for User Input
- Conditional Statements
- Switch Case
- Basic Arithmetic Operations

## 🚀 How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/chiragsinghcs223/Hotel_Billing_System.git
```

### 2. Navigate to the Project Directory

```bash
cd Hotel_Billing_System
```

### 3. Compile the Java Program

```bash
javac HotelBillingSystem.java
```

### 4. Run the Program

```bash
java HotelBillingSystem
```
